# Sweep Gateron (JST-TRS)
Adds a 2-pin JST connector in the same position as the TRS connector. This can be used for wired or wireless builds. The JST connector can be purchased [here](https://typeractive.xyz/products/battery-jack).

## Build Instructions
See the [Sweep High](https://github.com/davidphilipbarr/Sweep/tree/main/Sweep%20Bling%20MX) folder for PCB manufacturing instructions.
